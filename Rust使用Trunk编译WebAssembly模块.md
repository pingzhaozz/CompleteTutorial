## 安装`Trunk`
    cargo install trunk
    cargo install --locked wasm-bindgen-cli
## 添加`target`
    rustup target add wasm32-unknown-unknown
## 创建项目
    cargo init 项目名
## 编译运行
    trunk serve --open
